# 🚀 Yahoo Finance 버전 - 간단 설치 가이드

**API 키 없이 바로 시작 가능!** 5분이면 완료됩니다.

---

## ⚡ 빠른 시작 (3단계)

### 1️⃣ GitHub에 업로드

1. [GitHub.com](https://github.com) 로그인
2. 우측 상단 "+" → "New repository"
3. 이름: `stock-recommendation` (원하는 이름)
4. Public 선택 → "Create repository"
5. 모든 파일 드래그 앤 드롭으로 업로드

### 2️⃣ Vercel로 배포

1. [Vercel.com](https://vercel.com) 접속
2. "Continue with GitHub" 클릭
3. "New Project" → GitHub에서 방금 만든 저장소 선택
4. **환경 변수 설정 불필요!**
5. "Deploy" 클릭!

### 3️⃣ 완성! 🎉

- 1-2분 후 배포 완료
- Vercel이 제공하는 URL 확인
- 모바일/PC에서 접속하여 바로 사용!

---

## 📱 사용 방법

1. **웹 사이트 접속**
   - Vercel이 준 URL 접속
   - 예: `https://your-project.vercel.app`

2. **조건 설정 (기본값 그대로 OK)**
   - POC 오차범위: 2%
   - 반등 강도: 1%
   - 거래량 배수: 1.5배

3. **검색 시작**
   - "종목 검색 시작" 버튼 클릭
   - 10-20초 대기

4. **결과 확인**
   - 매수/매도 추천 종목 리스트
   - 각 종목의 근거 확인

## ❓ 자주 묻는 질문

### Q: API 키가 필요한가요?
**A:** 아니요! Yahoo Finance는 API 키 없이 무료로 사용 가능합니다.

### Q: 비용이 드나요?
**A:** 완전 무료입니다. (GitHub 무료 + Vercel 무료 + Yahoo Finance 무료)

### Q: 정확한가요?
**A:** Yahoo Finance 데이터를 사용하며, 약간의 지연이 있을 수 있습니다. 더 정확한 데이터를 원하시면 한국투자증권 API로 업그레이드 가능합니다.

### Q: 실제 매매가 가능한가요?
**A:** 이 버전은 시세 조회 및 분석만 가능합니다. 실제 매매는 한국투자증권 API 버전으로 업그레이드 시 가능합니다.

### Q: 종목이 하나도 안 나와요
**A:** 조건을 완화해보세요:
- POC 오차범위: 3-5%로 증가
- 거래량 배수: 1.2배로 감소
- 가격 범위 확대

## 🎯 사용 팁

### 최적의 검색 시간
- **장 마감 후 (15:30~)**: 당일 데이터로 내일 전략
- **아침 (09:00 전)**: 오늘 주목 종목 확인

### 조건 조정
- **공격적**: POC 3%, 반등 0.5%, 거래량 1.2배
- **보수적**: POC 1%, 반등 2%, 거래량 2배

### 결과 해석
- 🟢 **매수 시그널**: POC 지지 + 거래량 증가 = 상승 가능성
- 🔴 **매도 시그널**: POC 저항 + 거래량 증가 = 조정 가능성
- ⚠️ 반드시 본인 판단 필요!
